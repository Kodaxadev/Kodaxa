## The problem

Operational knowledge in a living world is scattered across memory, chats, screenshots, browser tabs, and reports that may no longer be current.

## What was built

Signal Vault records structured Signals linked to objects, systems, routes, and tribes, with confidence, source context, staleness behavior, and contradiction warnings. Its product surfaces include a browser app, supported in-game object route, desktop companion overlay, and an optional remote API path.

## Boundary

Signal Vault stays local-first by default. It does not perform process injection, does not silently infer verified object truth from hints, and does not claim production remote writes until deployed-role verification is complete.
