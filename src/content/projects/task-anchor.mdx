## The problem

AI-assisted work can rapidly drift from the task a user meant to finish, especially when new ideas continuously surface inside a coding session.

## What was built

Task Anchor introduces task locks, drift detection, scoped edit validation, explicit completion evidence, parked-idea queues, session checkpoints, and configurable interaction tone for focus-sensitive workflows.

## Boundary

Task Anchor supports human-directed focus and continuity. It does not replace the user's judgment about which task matters or whether priorities should change.
