## The problem

Plain-text diffs create noise when tabular files are reordered or columns change.

## What was built

RowDiFF performs key-aware row comparison, column rename detection, type classification, statistical summaries, semantic merge strategies, and terminal, JSON, or HTML reporting.

## Boundary

RowDiFF is presented as a technical tool for reviewable data changes. Its conclusions depend on configured keys, schema rules, and checks.
