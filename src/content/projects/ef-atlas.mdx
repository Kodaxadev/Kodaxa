## The problem

Builders and AI agents need to search distributed Frontier documentation while distinguishing official truth from community reference material.

## What was built

EF-Atlas ingests official documents, tooling repositories, API specifications, and carefully labeled community references into a searchable corpus. Each record retains an authority tier and source metadata, with web browsing, export, and agent-oriented context endpoints.

## Boundary

Authority tiers are a provenance control, not a claim that community records are canonical. Source hierarchy remains visible so users and agents can decide what evidence supports a conclusion.
