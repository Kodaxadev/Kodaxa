## The problem

Project context and prior decisions vanish when they remain trapped inside temporary work sessions.

## What was built

ChronosMCP supplies a local memory layer and a structured project graph with recall, historical snapshots, dependency tracking, and task suggestion tooling. It exists to keep decisions and project state durable outside transient context.

## Boundary

Chronos is described as a local project-memory tool. It is not represented as an autonomous decision maker or a replacement for source documentation and verified project state.
